# Contribution Guidelines

Thank you for your interest in contributing to the AirDrawingAI project! These guidelines are designed to provide you with a clear understanding of how you can contribute effectively.

## Getting Started
1. **Fork the Repository**: Click the 'Fork' button on the top right of this page to create your own copy of the repository.
2. **Clone Your Fork**: Use the command `git clone https://github.com/YOUR_USERNAME/AirDrawingAI.git` to download your copy locally.
3. **Create a Branch**: Before making changes, create a new branch using `git checkout -b your-feature-name`.

## Making Changes
- Make your changes and ensure they are in line with the project's coding style and standards.
- Write clear, concise commit messages that explain your changes.

## Submitting Your Contribution
1. **Push Your Changes**: Use `git push origin your-feature-name` to push your changes to your fork.
2. **Create a Pull Request**: Go to the original repository and click on 'Pull Requests', then click on 'New Pull Request'. Select your branch from the dropdown.

## Code of Conduct
Please adhere to our code of conduct in all interactions relating to the project.

Thank you for contributing!
