## 🚀 Features
- ✋ Hand tracking using MediaPipe
- 🎨 Draw with your index finger
- 🎯 Color selection using gestures
- 🧽 Eraser mode
- 📸 Screenshot capture
- ⚡ Real-time performance (FPS optimized)

---

## 🎮 Controls

| Gesture | Action |
|--------|--------|
| ☝ Index finger | Draw |
| ✌ Index + Middle | Select color |
| ✊ Fist | Clear screen |
| 🖐 Open hand | Screenshot |
