# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup
- Hand tracking using MediaPipe
- 8 colors + eraser functionality
- Adjustable brush size controls
- Gesture-based controls (draw, select, clear, screenshot)
- Screenshots functionality (canvas + combined view)
- Modern dark theme UI
- Troubleshooting documentation
- Contributing guidelines

### Features
- Draw in the air with finger gestures
- Real-time hand tracking and detection
- Smooth tip tracking with motion smoothing
- Gesture debouncing to prevent accidental triggers
- Keyboard controls for settings adjustment
- FPS monitoring and performance tracking
- Detection confidence adjustment

## [1.0.0] - 2026-03-18

### Initial Release
- Full application implementation
- OpenCV integration for camera input
- MediaPipe for hand landmark detection
- Real-time drawing canvas
- Color palette selection
- Screenshot capture with timestamps
- Comprehensive README documentation
- MIT License
